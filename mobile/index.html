Beginner Friendly Repositories

Here are some cool projects that you can contribute to to get started:

  • Termux Application: Termux is a terminal emulator for Android OS. They are also one of the most popular open-source Android packages.

  • android-gif-drawable: This open-source project is used for created Views and Drawable for displaying animated GIFs on Android.

  • Anki-Android: You have probably used flashcards to help you retain information for an exam or when learning a new language. AnkiDroid is used for creating flashcards on Android and is a perfect project to contribute to if you want to give back to the community. They even have a really nice contributing guide to help you get started!

  • Lawnchair: This is popular open-source home app for Android. Look at some of their good first issue labelled issues to know where to get started!

  • Traccar: This is a Java based open-source GPS tracking system. They have apps for both Android and iOS, so you can contribute whether you know Kotlin or Swift!

The app is 100% SwiftUI, uses the Twitter API, and AWS Amplify DataStore. See the README file to get started.
