Beginner's checklist for Hacktoberfest

Posted by Women Who Code on October 07, 2021 · 1 min read

If you are excited for the Hacktoberfest let us drive into a quick checklist you can follow as beginners


Sign up to get a GitHub or GitLab account.

Sign up on the Official Hacktoberfestwebsite. Make sure to do it between October 1st and October 31st

Search for ‘hacktoberfest’ topic to know if a project is participating in Hacktoberfest

Read through the code carefully and also look through the issues. It will have important clues on where you can do a PR

Make a Pull Request to the repository with the hacktoberfest tag

You need to make four commits with the "hacktoberfest" tag by October 31st 2021

If your commit is tagged as invalid or spam, then it won’t be counted towards Hacktoberfest

Don’t know where to start?

Check out the Technical Stack Repositories on this website! Start with the stack you know well and go through the beginner friendly repositories.
